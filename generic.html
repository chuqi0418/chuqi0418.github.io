Minecraft 下载指南

官方启动器、第三方启动器、服务器端及模组包的下载方式与推荐。

Java 版与基岩版

Java 版（Minecraft: Java Edition）是 PC 平台的经典版本，支持广泛的模组和自定义内容。可通过官方启动器购买并下载，支持 Windows、macOS、Linux 系统。
基岩版（Minecraft: Bedrock Edition）则跨平台支持移动端、主机和 Windows 10/11，具有更好的性能优化和联机体验。两者存档不互通，请根据需求选择。

官方下载地址请访问 Minecraft 官网。国内玩家也可通过各应用商店或授权平台获取基岩版。如需离线游玩或安装模组，推荐使用第三方启动器（见下节）。

推荐启动器与模组

HMCL（Hello Minecraft! Launcher）是一款国人开发的免费开源启动器，支持多版本隔离、模组一键安装、自动补全依赖等便捷功能。
PCL2（Plain Craft Launcher 2）同样深受玩家喜爱，界面美观且内置下载源，可快速下载游戏本体及模组。
下载地址：https://pcl.ruanmao.net/
备用链接（当前者无法打开时使用，可能失效）：蓝奏云网盘
MultiMC 则适合高级玩家，提供强大的实例管理能力。安装模组时，可配合 CurseForge 或 Modrinth 平台搜索海量模组包。

官方启动器

从 Minecraft 官网下载正版启动器，登录微软账户即可畅玩 Java 版和基岩版。
